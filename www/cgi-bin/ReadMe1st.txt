Calamity Jane Express(R) simple write file script by Dee Jay Alexander at www. calamityjaneexpress.net.
Open source GPL2+ license.
These files must be in some cgi-bin file.
All files must be read and written to, except cje.sh. This script file must have read and executable rights.
And it must also be able to change the file cjea.xml and cjeb.xml in the documentroot (e.g. /var/www) of the server. 
See the script cje.sh.
